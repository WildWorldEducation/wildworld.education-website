Train4Football.com

Web application to help football/soccer players improve reaction speed

Screenshots

Project Overview

  • Purpose: To test and improve reaction times of soccer players through engaging web-based minigames.

  • Description: A web application with minigames, user accounts, and a payment system for reaction speed training.

Client

Mellow Yellow

Key Features

  • Reaction Time Minigames: Interactive games designed to enhance soccer players' reflexes and decision-making skills.

  • User Account System: Personalized profiles for tracking progress.

  • Payment Integration: Secure payment system.

Link

  • Live Application

    • Tech Stack

    • Frontend: Vue.js for a dynamic and interactive user interface.

    • Backend: Express and Node.js for robust server-side operations.

    • Database: MariaDB for efficient data management.

    • Game Development: Phaser.js for creating engaging web-based minigames.

    • Cloud Infrastructure: AWS for scalable hosting and deployment.

    • Payment Processing: Stripe integration for secure transactions.
