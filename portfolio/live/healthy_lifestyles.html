Healthy Lifestyles - Going On Tour

Web application with games, helping students learn about being healthy.

Screenshots

Project Overview

  • Purpose: To teach kids about healthy lifestyles through interactive web games and educational content.

  • Description: Web application designed for teachers and students featuring engaging web games on healthy living topics.

Client

Rad Schools

Key Features

  • Interactive Web Games: Fun, educational games that teach healthy lifestyle concepts.

  • Teacher Dashboard: Basic way for educators to track student progress.

  • Educational Content: Comprehensive lessons on nutrition, exercise, and wellness.

  • Responsive Design: Accessible on various devices for flexible learning environments.

Link

  • Live Application

Tech Stack

  • Backend: Node.js and Express for server-side logic and API development.

  • Frontend: Vue.js for a dynamic and responsive user interface.

  • Database: MariaDB for reliable data storage and management.

  • Cloud Infrastructure: AWS for scalable hosting and deployment.

  • Game Development: Phaser.js for creating interactive web games.
