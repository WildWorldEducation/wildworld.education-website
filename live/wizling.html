Project Overview

  • Purpose: To tutor young kids through an engaging, interactive AI-powered platform.

  • Description: Speech to speech AI tutoring web application designed for children.

Client

Collins Institute

Key Features

  • Speech-to-Speech Interaction: Real-time voice communication between students and AI tutors.

  • AI-Powered Tutoring: Intelligent AI that adapts to each child's learning pace and style.

  • Kid-Friendly Interface: Engaging and intuitive design tailored for young learners.

Tech Stack

  • Backend: Node.js and Express for robust server-side operations.

  • Frontend: Vue.js for a dynamic and responsive user interface.

  • Database: MariaDB for efficient data management.

  • Cloud Infrastructure: AWS for scalable deployment and hosting.

  • AI Integration: OpenAI Realtime API for advanced speech-to-speech capabilities.
