VisibleBottleneck.org

Test your multitasking

Project Overview

  • Purpose: To showcase Professor Hal Pashler's research on attention and multitasking limitations.

  • Description: An interactive web application featuring minigames designed to test and demonstrate attention bottlenecks and multitasking challenges.

Client

Hal Pashler - UCSD

Key Features

  • Attention Testing Minigames: Interactive games that measure cognitive processing limits and attention bottlenecks.

  • Multitasking Challenges: Scenarios that demonstrate the difficulties of performing multiple tasks simultaneously.

  • Educational Insights: Explanations of psychological research findings on attention.

  • Accessible Design: User-friendly interface suitable for both academic study participants and general visitors.

Tech Stack

  • Backend: Node.js and Express for server-side logic and data handling.

  • Frontend: Vue.js for a responsive and interactive user interface.

  • Database: MariaDB for storing user data and research results.

  • Cloud Infrastructure: AWS for scalable hosting and deployment.

  • Game Development: Phaser.js for creating engaging attention-testing minigames.
